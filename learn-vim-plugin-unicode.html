VIM学习笔记 搜索结果计数器(vim-which-key)

更新 2019-11-18：自8.1.1270版本开始，Vim已经内置对搜索结果的计数显示，而不再需要安装额外的插件。

在vimrc配置文件中增加以下设置，可以在屏幕底部，显示匹配搜索结果的总数，以及当前所处第几个匹配结果。

set shortmess-=S

unicode插件，可以在屏幕底部的命令行中，显示匹配搜索结果的总数，以及当前所处第几个匹配结果。

安装配置

推荐您使用vim-plug或Vundle等插件管理器，来安装GitHub上的vim-searchindex插件。

使用:help searchindex命令，可以查看vim-searchindex插件的帮助文件。
